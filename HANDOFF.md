# BHCC — состояние на 16 июля, вечер

Запушено в `main` → https://github.com/Sigovs/BHCC. Открывай `index.html`.

## Что решено окончательно (не пересматривать)

- **Шрифты:** Gloock 400 (display, заголовки секций, заголовки карточек) · Newsreader italic 400 (только цитаты §07 — у Gloock нет курсива) · Inter Tight (тело + UI, вес 450 на тёмном фоне). Система 84.9 KB latin. Перебор из девяти шрифтов закрыт — вернулись к этой связке.
- **Hero:** вариант «C» — H1 по левому краю, «by Alex Manos» курсивом встык после «Car Club», кнопки под подзаголовком слева. Без фолда на десктопе (100vh), мобильный — 70vh по брифу.
- **Видео:** живое, `assets/video/hero-loop.{webm,mp4}` (1.38 / 2.20 MB, 7.97 c, бесшовный луп). Постер = кадр 0 лупа, старт без прыжка. Мастер `home.mp4` (49 MB) под `.gitignore` — в git не класть.
- **Палитра, отступы, radius 0, Bootstrap 5.3.8** с тремя вычищенными багами — в `tokens.css` / `_variables.scss`, готово.

## Осталось доделать (по убыванию важности)

1. **Хедер поверх кадра.** Сейчас `sticky` с плашкой, hero подрезан под него. В твоём макете лого + nav лежат ПОВЕРХ изображения без фона. Нужно: прозрачный поверх hero в покое → off-white с тенью на скролле (бриф §01 это разрешает — «shadow on scroll only»). Следствия: hero станет полной 100vh; nav поверх светлого кадра (яркость 84–137) — вероятно белый текст + лёгкий градиент сверху; проверить контраст замером.

2. **Hero на 1366×768 и 1280×720:** лок-ап пересекает нос машины. Агент это заметил и собирался замерить положение машины на t=4.0s, но был остановлен. На больших экранах всё чисто. Нужно доправить кегль/позицию под низкие ноутбуки — типографика должна отвечать на высоту, не только на ширину.

3. **Картинки блога негодные** — `assets/images/blog/post-{1,2,3}.jpg`: пропорции 1.38–1.42 вместо 16:9, и на post-2/post-3 ВЫЖЖЕН чужой текст (заголовки, подкаст-плашка). Скрейп чужих превью. Заменить на чистые фото машин 16:9 (можно из `assets/images/inventory/`).

4. **Логотипы прессы** — файлов нет, стоят текстовые заглушки (GOOGLE / YELP / HEMMINGS / …). Нужны официальные марки от клиента (у Google/Yelp brand guidelines ограничивают grayscale) либо оформить заглушку как намеренную.

## Как запускать

Статика, без сборки — открыть `index.html` двойным кликом или `python3 -m http.server`. SCSS-слой (`assets/scss/`) собирается в Bootstrap-CSS, но для просмотра не нужен — готовый CSS в `assets/css/`.

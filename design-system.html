Beverly Hills Car Club · by Alex Manos

Design system

The spec for the homepage — type, colour, spacing. Every value below is a live token from assets/css/tokens.css; change it there and it updates here. Contrast ratios are measured (WCAG 2.1), not estimated.

01

Typefaces — three families, by role

Aa

Gloock — display

Hero, section headings, card titles. One static weight (400), lining figures. The loud didone that carries the brand voice.

Aa

Newsreader italic — quotes

§07 testimonial quotes only. Gloock has no italic; a synthesised oblique on a didone is unacceptable, so a true italic serif is loaded as a micro-subset.

Aa

Inter Tight — body & UI

All running text and interface. Neutral under the didone; unambiguous figures for price, year and phone. Weight 450 on dark surfaces (measured).

The champagne gold was retired (2026-07). A neutral has no hue to carry a mid tone, so the accent splits by surface, not by lightness: --accent near-black on light, --accent-dark near-white on dark. There is no value that works on both — a mid grey measures ~1.2 on one of them. Red stays what it always was: content (the shield, the hero car), never interface.

04

Buttons — two types: primary & secondary

Two types only. Primary (main) — a solid ink pill; on a dark surface it inverts to a cream fill with ink text (14.76 ✓). Secondary — transparent + a neutral hairline; cream text on dark. Pill radius is the art-director exception for interactive controls — the system radius stays 0 for cards and structure. CTAs are sentence case per the brief, not tracked caps.

No border, no shadow — the brief bans both on inventory cards. Year + make + model in Gloock (2-line box, ceiling 20px so the longest real title stays inside); price in mono like an odometer reading; hover reveals a slight scale (1.03), it lifts nothing.
